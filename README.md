# CurrencyApp

## About

Project that displays over 150 currencies rates in comparison with EURO. It supports landscape mode and can load currencies infinitely.

## Technologies used

- Kotlin
- Coroutines
- Jetpack Compose
- Dagger Hilt
- Retrofit
- Junit4
